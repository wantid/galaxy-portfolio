WebGL is not available

This application requires WebGL to run. Your browser or device does not support WebGL or it is disabled.

How to fix:

  • Enable hardware acceleration: Go to browser settings → System → Enable "Use hardware acceleration"
  • Check WebGL status: Open chrome://gpu in Chrome or about:support in Firefox
  • Update GPU drivers: Install the latest drivers for your graphics card
  • Try a different browser: Chrome, Firefox, or Edge usually have good WebGL support
