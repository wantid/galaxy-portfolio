# Газпром автоматизация

**Инженер-программист 1 категории**

[gazauto.gazprom.ru](https://www.gazauto.gazprom.ru/)

---

Разработка переиспользуемых решений и инструментария для промышленных систем автоматизации.

## HMI Development (Weintek / EasyBuilder Pro)

Создал универсальный фреймворк для разработки интерфейсов операторских панелей:

- **Template Engine** — типовой шаблонный проект, ускоряющий разработку новых интерфейсов
- **Auth Module** — модуль авторизации и блокировки панели с разграничением прав доступа

## PLC Programming (SCADA Текон / ST)

Разработал библиотеку переиспользуемых функциональных блоков на языке Structured Text:

- **AVR & Rotation Block** — универсальный блок управления исполнительными механизмами с поддержкой режимов автоматического включения резерва и ротации
- **Industrial Templates** — набор типовых алгоритмов для стандартных объектов автоматизации
