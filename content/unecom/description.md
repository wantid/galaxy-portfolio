Разработка инструментария и переиспользуемых модулей для платформы iRidium (Smart Home).

## iRidium Studio Toolchain

Создал JavaScript-фреймворк для ускоренной разработки UI панелей управления умным домом:

- **Auto-Build System** — автоматическая сборка и адаптивная вёрстка под любые устройства
- **Excel Parser** — импорт конфигураций и данных из Excel-таблиц
- **Modular Architecture** — модульная структура для переиспользования компонентов

## iRidium Server Modules

Разработал библиотеку универсальных серверных модулей на JavaScript:

- **Scene Engine** — централизованный модуль управления сценами и сценариями с расширяемой архитектурой
- **Scheduler Module** — система пользовательских расписаний и автоматизаций
- **Leak Detection System** — модуль мониторинга датчиков протечек
