# Last Shift

A Telegram bot game written during mandatory military service under limited conditions.

## Idea

Most of the time on the phone is spent on Telegram, and besides that — games. This is how the idea to write a game with limited resources was born.

**Last Shift** is a turn-based 2D clone of Lethal Company with multiplayer.

## Features

- Turn-based gameplay
- Multiplayer via Telegram
- Visibility area
- Interactive objects and tools
- Lobby system

## Stack

- **Node.js** + **Express** — backend
- **Telegraf** — Telegram Bot API
- **PostgreSQL** — database
- **Docker Compose** — containerization
- **Jimp** — image generation
- **dungeon-generator** — procedural level generation

## Development Tools

- **Termux** — Linux terminal emulator for Android
- **ACode** — code editor similar to VSCode
- **Remote server** — for database
